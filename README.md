# Gostou, Comprou, Chegou - Website

Site institucional para serviço de compras e entregas rápidas.

## Tecnologias
- HTML5 Semântico
- CSS3 (Variáveis, Flexbox, Grid)
- JavaScript Vanilla (Sem frameworks)
- Feather Icons (Ícones SVG leves)

## Como Editar
1. As cores globais estão no início do arquivo `style.css` na variável `:root`.
2. Os textos e seções estão organizados no `index.html`.
3. O script do menu mobile está em `script.js`.

## Deploy
Este projeto está pronto para ser hospedado no GitHub Pages, Netlify ou Vercel.
